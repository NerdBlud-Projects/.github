# Nerdblud-Projects

Nerdblud-Projects is a group that works together on software projects. This space is for projects that're not personal and are meant to help solve real problems or provide tools for the community. 

Unlike projects the work here is owned by everyone and is meant to be useful to others. Every project here was built to help someone or to explore a solution with other developers.

---

### Our Focus
The projects in this group are divided into two categories:

* **Community Tools:** These are small to medium-sized tools that fill the gaps that developers have in existing workflows or provide free alternatives to common problems which is very useful and most likely what people are looking for.
* **Collaborative Experiments:** These are projects where many people have worked together.

### Collaboration
This group is based on the idea that software's better when many people work on it together. We focus on:

* **Code:** We write code that is easy for humans to understand.
* **Practicality:** We build things that're useful and solve real issues.
* **Openness:** We encourage others to improve our work.

### Projects
You can find a list of projects at **Repositories**. Each project has its own documentation about their usage. 

If you have worked on a project here or want to suggest a project that helps people through rough times feel free to reach out by making a pull request or opening a discussion to let us know!

### Standards
Even though these are projects we keep high standards, for commit history, documentation and code quality. We think that shared projects need attention to detail because many people will use and understand them.
